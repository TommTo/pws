Behind the scenes

Wij nemen nu een kijke achter de schermen van de Websites. HTML (Hypertext Markup Language) en CSS (Cascading Style Sheets) zijn twee van de belangrijkste technologieën die worden gebruikt bij webontwikkeling. In de vroege jaren van de Web werd voor de grootste HTML gebruikt voor alles en kort voor de introductie van CSS werd JavaScript populair. CSS werd voor het eerst geïntroduceerd in 1996 als een manier om de presentatie van een webpagina te scheiden van de inhoud ervan.

De eerste versie van CSS is ontworpen om een ​manier te bieden om de visuele weergave van HTML-elementen te regelen. Door de visuele en de structuur van een website te scheiden werd de code overzichtelijker en de files kleiner. (Kijk naar de slideshow rechtsboven)

Tegenwoordig kan je niet om CSS heen als je een webdeveloper wil worden.
